Excel DataLink Example

This example shows how to extract and insert records from/to a Microsoft Excel Worksheet.

Introduction

If You want to transfer data from this Excel to a File or other Data Storage you must follow this steps:

For example with the following worksheet (from the Northwind DB)

 

To use Delimited Format

You must define:

	[DelimitedRecord("|")]
	public class CustomersVerticalBar
	{
		public string CustomerID;

		public string CompanyName;

		public string ContactName;

		public string ContactTitle;

		public string Address;

		public string City;

		public string Country;
	}

The Solution

   Finally use the ExcelDataStorage to Read the records. 

							
   ExcelStorage provider = new ExcelStorage(typeof(CustomersVerticalBar));
   
   provider.StartRow = 3;
   provider.StartColumn = 2;
			
   provider.FileName = @"..\data\Customers.xls";
   
   CustomerVerticalBar[] res = (CustomerVerticalBar[]) provider.ExtractRecords();
