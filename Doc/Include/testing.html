Testing

This library has been developed using agile and test driven development.

A regression test suite is run whenever any part of the library is changed.

There are more than 420 NUnit Tests !!

I want to give thanks to Jamie Cansdale for the excellent TestDriven.net

If you don't use it yet, you must check it out, it's fantastic!!!

Remember: If you add functionality to the library, don't forget to run these tests to verify that nothing is broken.

If you find a bug in the library, write a test to show the bug, then fix it. Otherwise how will you know it has been fixed and will stay fixed.
