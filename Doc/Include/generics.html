 .NET 2.0 Generics Support

From the version 1.3.5 and foward the FileHelpers supports the use of Generics for the Engines.

The names of the classes are the same that without generics, then you can use both:

C# 2.0

Standard Way 
FileHelperEngine engine = new FileHelperEngine(typeof(Customers));

Generics Way

FileHelperEngine<Customers> engine = new FileHelperEngine<Customers>();

VB.NET 2.0

Standard Way
Dim engine As New FileHelperEngine(GetType(Customers))

Generics Way
Dim engine As New FileHelperEngine(Of Customers)();

The main diference is that you don´t need to cast the engine results:

Standard Way 
Customers[] res = (Customers[]) engine.ReadFile("File.txt");

Generics Way
Customers[] res = engine.ReadFile("File.txt");
